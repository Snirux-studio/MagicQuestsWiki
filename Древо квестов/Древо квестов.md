Это меню, которое открывается при помощи предмета [[Старая книга]], путем нажатия на ПКМ держа предмет в руке. Древо квестов, как понятно из названия, представляет из себя древо, которое состоит из сегментов.

### Сегменты

**Сегменты** - это некоторые области древа из нескольких веток из пары десятков квестов. По умолчанию в книге открыт только один - [[Сегмент Начало]]. Для открытия остальных необходимо вставить в книгу [[Модули для старой книги]]. Как правило, квесты в сегменте орентированы на одну и ту же часть мода.
